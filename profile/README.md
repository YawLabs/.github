# Yaw Labs

Developer tools for the terminal, AI, and cloud infrastructure.

---

### Products

**[Yaw Terminal](https://yaw.sh)** -- Cross-platform terminal emulator with built-in AI (9 providers), SSH/database connections (6 types), and a file editor. Free, zero telemetry.

**[Token Meter](https://tokenmeter.sh)** -- LLM spend tracking, smart routing, and budget alerts across 10 providers. MCP server + API gateway + dashboard.

**[mcp.hosting](https://mcp.hosting)** -- MCP server compliance testing, proxy, and hosting platform. Free compliance tester with 24 tests. [Self-hosted deploy templates available](https://github.com/YawLabs/mcp-hosting-deploy).
