# Contributing to Yaw Labs

Thanks for your interest in contributing to Yaw Labs projects!

## Where to Contribute

This repo contains the Yaw Labs organization profile. For code contributions, head to the specific project repo:

- [ssh-mcp](https://github.com/YawLabs/ssh-mcp) — SSH operations MCP server
- [tailscale-mcp](https://github.com/YawLabs/tailscale-mcp) — Tailscale MCP server
- [electron-mcp](https://github.com/YawLabs/electron-mcp) — Electron.js MCP server
- [npmjs-mcp](https://github.com/YawLabs/npmjs-mcp) — npm registry MCP server
- [caddy-mcp](https://github.com/YawLabs/caddy-mcp) — Caddy MCP server
- [lemonsqueezy-mcp](https://github.com/YawLabs/lemonsqueezy-mcp) — LemonSqueezy MCP server
- [mcp-connect](https://github.com/YawLabs/mcp-connect) — MCP orchestrator
- [mcp-compliance](https://github.com/YawLabs/mcp-compliance) — MCP spec compliance testing
- [ctxlint](https://github.com/YawLabs/ctxlint) — AI context file linter
- [electron-optimize](https://github.com/YawLabs/electron-optimize) — Electron optimization utilities

Each project has its own CONTRIBUTING.md with setup instructions.

## General Guidelines

- All PRs require maintainer approval before merging
- Direct pushes to default branches are blocked
- One focused change per PR

## For AI Agents

Fork the target repo, create a branch, run lint and tests, and submit a PR with a clear description. See the target repo's CONTRIBUTING.md for project-specific commands.
